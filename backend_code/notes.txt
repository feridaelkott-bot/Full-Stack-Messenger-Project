Setting up Javalin with Maven

--> What is Maven? 
Maven is a build automation tool that is used for Java projects. 
It addresses two aspects of building software: 

1. It describes how the software is built, and 
2. It describes the software's dependencies

Maven projects are configured using a Project Object Model, which is stored in a pom.xml-file

--> What is a Project Object Model exactly? 
A POM provides all the configuration needded for a single project. 
this includes the name, and its dependencies on other projects. 

A POM can also configure the individual phases of the build process, adn tehse are implemented as plugins. 
For example, you can condigure the compiler plugin as using Java version 1.5 or specify packaging the project even if some unit tests fail. 

here is a simple example of a pom.xml file used for our project's configuration: 

<project>
    <!-- model version - always 4.0.0 for Maven 2.x POMs -->
    <modelVersion>4.0.0</modelVersion>

    <!-- project coordinates - values which uniquely identify this project -->
    <groupId>com.mygroup</groupId>
    <artifactId>my-javalin-project</artifactId>
    <version>1.0.0-SNAPSHOT</version>

    <!-- library dependencies: here, the dependencies show that our project is working with Javalin -->
    <dependencies>
        <dependency>
            <groupId>io.javalin</groupId>
            <artifactId>javalin</artifactId>
            <version>6.7.0</version>
        </dependency>
    </dependencies>
</project>



*NOTE: depending on your IDE, (Integrated Development Environment), there will be different ways in which these dependencies and configurations are implemented. 



In a nutshell: The Project Object Model is the fundamental unit of work in Maven
-Maven is a project management tool that operates entirely based on the information defind in the Project Object Model


--> So basically, whenever you use Maven to compile your project or execute any command, it first reads the pom.xml file to ensure it performs the requested action based on teh Project Object Model's requirements
